# My Second Brain 🧠

¡Dímelo! Esta es una nota de prueba.

Una vez que configures tu script de exportación en Obsidian, todas tus notas convertidas a Markdown estándar caerán en esta carpeta (`content/brain/`), y Nuxt Content se encargará de renderizarlas automáticamente con el plugin de Typography de Tailwind.

## ¿Qué sigue?

1. Escribe en Obsidian con normalidad.
2. Usa un plugin como **Obsidian Export** o un script de Node/Python para limpiar los `[[WikiLinks]]` y pasarlos a `[Links Markdown](./ruta)`.
3. Apunta el destino de esa exportación directo a la carpeta `content/brain/` de este proyecto.

¡Y ya 'tamos ready!
